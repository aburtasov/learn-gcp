1. gcp-organization
Высший компонент Иерархии GCP 
Доступ на этом уровне в IAM распространяется на все ниже описанное(Уровень Бога)
Может состоять из Project или Folder 

2. gcp-folder 
Доступ на этом уровне в IAM распространяется на все Folders и Projects  внутри
Может состоять из Folder или Project

3. gcp-project 
Доступ на этом уровне в IAM распространяется только на этот Project
Может состоять из Resources

4. gcp-resources 
Нижний компонент Иерархии GCP, на некоторые ресурсы доступ можно давать на этом уровне
